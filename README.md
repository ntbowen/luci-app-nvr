Luci App NVR：让你的路由器秒变监控录像机
Luci App NVR 是一款运行在 OpenWrt/iStoreOS 路由器上的轻量级监控管理插件。只要你有摄像头，就能在路由器上实现录像、回看和远程监控，再也不用专门买昂贵的 NVR 录像机了。



🔥 为什么选择它？（核心功能）
省钱省心：直接利用软路由的剩余空间进行录像，支持外挂硬盘。
支持所有摄像头：只要摄像头支持 RTSP（市面 99% 的品牌都支持），就能接入。
中文界面：全中文操作菜单，一看就会用。
后台静默录像：路由器开机就自动开始录制，断电重启也不怕。
手机远程监控：配合“易有云”插件，出门在外随时随地看家里的监控回放。


📱 进阶：如何实现手机远程查看？
这是本插件最强大的玩法，只需两步：
安装易有云：在路由器的软件中心找到“易有云”并安装绑定。
开启穿透：通过易有云的手机 App 开启内网穿透功能。 这样你即便人在外地，只要手机有网，就能打开路由器界面查看到 NVR 的录像文件。


📖 摄像头地址填什么？（常用格式大全）
在插件设置里填入对应的地址即可，记得把 admin 和 password 改成你摄像头的密码：
海康威视：rtsp://admin:密码@IP地址:554/h264/ch1/main/av_stream
大华/乐橙：rtsp://admin:密码@IP地址:554/cam/realmonitor?channel=1&subtype=0
TP-LINK/水星：rtsp://admin:密码@IP地址:554/stream1
小米 (支持NAS版)：rtsp://admin:密码@IP地址:554/live/ch0
其他品牌：如果你的说明书没写，通常尝试 rtsp://admin:密码@IP地址:554/onvif1


📥 改进后：如何下载最新安装包？
现在你不需要自己编译代码了，我们已经配置好了“云端工厂”自动生产：
点击网页上方的 [Actions]。
看到带绿色对勾的任务，点进去。
在页面最下方的 Artifacts 找到压缩包下载。
下载后解压出 .ipk 文件，在路由器后台上传安装即可。


🛠 常见问题说明
安装报错？ 先运行 opkg update。
录像存哪？ 建议在设置里把路径改到你挂载的硬盘目录下，不要存路由器闪存里，防止存满重启。
